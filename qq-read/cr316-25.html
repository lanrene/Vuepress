CE316-25

软件安装

安装方法来自贴吧《QQ阅读器CR316刷机时问题》，由@xcheh提供。

前提条件

  • 电脑
  • Micro USB 数据线（需要有数据传输能力）
    Micro USB 图片

提示

部分线可能只有充电功能。拥有数据传输能力的数据线相对会更粗，将设备连接电脑会有提示音和弹窗提醒。

准备工作

  • 下载安装《搞机工具箱》

    搞机工具箱是 UP 主晨钟酱开发的一款利用 ADB 来控制手机的玩机工具。

  • 准备要安装的 APP

    把要安装 APP 的包名修改为 com.yuewen.xxx 格式才能绕过安装限制（其中 xxx 为自定义内容）。这里提供了部分软件，如果没有你需要的，可以使用《MT管理器》来修改。修改方法可以参考这里。TODO

    注

    不是所有修改过包名的应用都能成功运行，这时需要更换 APP 版本，重新修改包名安装测试，直到成功启动，并能良好运行。

安装步骤

1. QQ 阅读器打开 ADB 调试

依次点击下面按钮，开启 ADB 调试

我
设备管理
设备信息
连续点击 5 次 序列号
下一页
adb已关闭

提示

  1. 点击退出应用，即可退出调试页面。
  2. 如果不小心点击序列号超过 5 次，没有进入调试页面，点击下面的成功 或者 失败即可。

2. 用数据线连接电脑和 QQ 阅读器

3. 电脑打开《搞机工具箱》

  • 切换到软件管理选项卡，点击安装软件后选择要安装的 APP 即可完成安装。

    提示

    必须安装桌面应用E-Ink Launcher，才能正常打开安装过的软件。

  • 切换到显示设置选项卡，在显示密度填入 300，点击修改。（此操作修复软件显示异常）

ROOT 设备

警告

此操作极度危险，容易致使设备变砖，无法修复。不建议小白尝试。

ROOT 可以更好的掌控手机，获得更好的玩体验。比如更换字体等。

实现方法

利用脏牛漏洞（这里使用开源项目 GetRoot-Android-DirtyCow），获取临时 ROOT 权限。

然后通过 dd 命令提取 boot.img，以及刷入面具修补后的boot。

前提条件

  • 电脑
  • Micro USB 数据线（需要有数据传输能力）

准备工作

提示

ROOT 方法与软件安装步骤相关，如果没有过该操作，请浏览安装软件，并完成桌面软件 E-Ink Launcher 和 文件管理软件 MT 管理器 的安装。

  • 下载安装《搞机工具箱》

完成安装软件的同学已经完成该项准备

  • 下载面具 Magisk

  • 确保已经安装了桌面应用和文件管理应用

操作步骤

1. QQ 阅读器打开 ADB 调试

详见上面安装步骤

2. 用数据线连接电脑和 QQ 阅读器

3. 电脑打开《搞机工具箱》

切换到ADB终端选项卡，依次复制下面命令到输入框中。

获取临时 ROOT

cd /sdcard
curl https://raw.githubusercontent.com/j0nk0/GetRoot-Android-DirtyCow/refs/heads/master/G1tR0oT -s -o G1tR0oT

提取 boot.img

dd if=/dev/block/mmcblk0p21 of=/sdcard/boot.img

boot.img 位置参考 [原创] 已 root 设备中提取 boot.img - 行简

4. 安装面具

  • Magisk

提示

必须安装桌面应用E-Ink Launcher，才能正常打开安装过的软件。安装方法详见安装软件。

5. QQ阅读打开 Magisk

依次点击下面按钮

1fsd

6. 回到《搞机工具箱》

依次复制下面命令到输入框中。

获取临时 ROOT

cd /sdcard
curl https://raw.githubusercontent.com/j0nk0/GetRoot-Android-DirtyCow/refs/heads/master/G1tR0oT -s -o G1tR0oT

查看修补过的 boot 文件名

ll | grep '\.boot$'

将修补过的 boot 刷入

dd if=/sdcard/boot.img of=/dev/block/mmcblk0p21/fsdafasdf.img

注意

这里的 fsdafasdf.img 要替换成上面命令输出的名字。
